## Packages
- express: ```npm install express```
- nodemon: ```npm install nodemon```
- cors: ```npm install cors```
- dotenv: ```npm install dotenv```
- node-postgres: ```npm install pg```

## Running the dev server
- ```nodemon server.js```

## Required Environment Variables
- PORT 
- POSTGRES_USER
- POSTGRES_PASSWORD
- POSTGRES_HOST
- POSTGRES_PORT
- POSTGRES_DATABASE
